O Que e Voice RAG?

Arquitetura Voice RAG no N8N

1. Webhook de Voz

Retell/VAPI envia webhook quando usuario fala. Payload inclui transcricao, session_id e metadata.

2. RAG Search

Transcricao vira query no vector store. Busca chunks relevantes da base de conhecimento.

3. LLM com Contexto

GPT-4 ou Gemini gera resposta conversacional usando chunks + historico da conversa.

4. Text-to-Speech

Resposta e convertida em audio via ElevenLabs e enviada de volta para o caller.

5. Guardrails

Validacao de respostas para evitar informacoes incorretas. Escalacao automatica para humanos quando necessario.
