O Que e Twilio?

Twilio e a plataforma lider em comunicacao programatica. Permite enviar SMS para qualquer numero do mundo, fazer chamadas automatizadas, WhatsApp Business e verificacao por codigo.

SMS

Envie mensagens para qualquer pais

WhatsApp

Alternativa ao Meta Cloud API

Voz

Chamadas automatizadas com TTS

Verify

2FA e verificacao por SMS/email

1

Criar Conta Twilio

Cadastro e creditos de teste

  1. 1. Acesse twilio.com e clique em Sign up
  2. 2. Preencha nome, email, senha e telefone
  3. 3. Verifique seu numero de telefone (recebera SMS)
  4. 4. Voce recebera $15 de creditos de teste

Trial Account

A conta trial permite enviar SMS apenas para numeros verificados. Para enviar para qualquer numero, faca upgrade (~$20).

2

Obter Credenciais

Account SID, Auth Token e Phone Number

No Console

  1. 1. No dashboard, va em Account → API keys & tokens
  2. 2. Copie o Account SID (comeca com AC)
  3. 3. Copie o Auth Token

Obter Numero de Telefone

  1. 1. Va em Phone Numbers → Manage → Buy a number
  2. 2. Escolha um numero (trial: 1 gratis)
  3. 3. O numero sera o remetente dos seus SMS
3

Configurar no N8N

Adicionar credencial Twilio

  1. 1. No N8N, va em Credentials → Add Credential
  2. 2. Busque por Twilio API
  3. 3. Preencha: 
Account SID: ACxxxxxxxxxxxxxxxxxxxxxxxx
Auth Token: seu-auth-token

Exemplos Praticos

1. Enviar SMS simples

// Twilio: Send SMS
From: +15551234567 // Seu numero Twilio
To: +5511999999999 // Destino
Message: "Ola! Seu pedido foi enviado."

2. Alerta de venda no SMS

// Stripe Trigger: payment_intent.succeeded
// → Twilio: Send SMS

To: +5511999999999
Message: "Nova venda! R$ {{ $json.data.object.amount / 100 }} de {{ $json.data.object.receipt_email }}"

3. Verificacao 2FA

// HTTP Request: POST Twilio Verify
URL: https://verify.twilio.com/v2/Services/VAxxxx/Verifications

// Body:
{
  "To": "+5511999999999",
  "Channel": "sms"
}

// Usuario recebe codigo de 6 digitos
